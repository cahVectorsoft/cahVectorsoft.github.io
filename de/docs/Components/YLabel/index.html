yLabel

Description​

The YLabel component serves as a powerful tool primarily designed for displaying short text information within your application. It's an adaptable element that allows both static and dynamic content.

With the YLabel you can display user instructions, show real-time data updates, provide context for other components, or present calculated values within a form, among other applications.

Some uses for the YLabel are:

  • SourceLink Input
  • SourceLink Table
tip

Use YTextarea or YTextbox for handling longer texts.

Basic Structure​

In the following the basic structure of the YLabel shall be explained. For the general Structure of a YComponent please visit the YBase-components basic-structure.

Properties​

Through its various properties the YLabel can be configured to suit your needs. The Display below provides you with an overview of all the YLabel properties.

Properties can be changed directly through three methods:

  • Inside the propertybar 1 and propertybar 2, which is positioned in your Workarea next to your component where you need it. It shows the most important properties, thus providing a fast and efficient way to configure your component in the most basic way.
  • It may also occur in the propertybar 2 which is a seamless extension accesible as a dropdown-item of the PB1. Extends the functionality of the PB1 by providing advanced Pickers for most used Properties.
  • Inside the propertybar 3, which is located in the righthand Drawer. Every Property of a component can be configured here.

Usage​

The following section provides detailed explanations of the various properties from the previous section and their usage. The headings and subheadings serve as the categorization into Categories and Groups. Only the properties that are relevant to are listed below. General properties that are applicable universally can be found in the YBase-component's usage-section.

Style > Label​

The Label can be used to show text inside the YLabel. You can only have one Label in a YLabel. If you want to customize it, you can do this with the TypographyPicker in propertybar 1 and propertybar 2 or under the Style Menu in the propertybar 3.

font-color​

ColorProperty

The color of the Label. This can be set as a color from a palette or a custom hex color. You can choose a color via the ColorPicker.

font-family​

ListProperty

The font-family of the Label. This can be set to a font-family from the specified list of fonts. The preselected font-family is "Quicksand".

font-weight​

NumberProperty

With the font-weight of the Label you can choose how bold your font should be. Normally font-weight is set in 100 steps. 100 is very thin, 900 very thick. The normal font-weight is 400.

font-size​

UnitNumberProperty

The font-size of the Label. This can be set in:

cm mm in px em ex ch rem

font-style​

ListProperty

The font-style of the Label. This can be set in:

normal italic oblique

text-transform​

ListProperty

You can transform the text of the Label with the text-transform property. Here you can set whether the text should automatically be written in upper or lower case and much more. The options are:

capitalize uppercase lowercase full-width none

font-align​

ListProperty

The font-align of the Label. Here you can determine whether the text should be aligned left-right or centered. This can be set in:

left center right

label​

StringProperty

With this property you can decide what should be the actual text that will be displayed in the label.

tip

The label text can also be set through the inplace-edit. Simply double click the components and the edit will appear.

Examples of use​

SourceLink Input​

You can establish a connection between the YLabel component and another component using aSourceLink-Picker. In this example, the YLabel component is linked to a YInput component. Whenever the value in the YInput is changed, the connected YLabel component will update accordingly.

How to Use​

  1. Create a YLabel component and a YInput in your application.
  2. Select the created YLabel component and open the propertybar 3.
  3. Within the settings, go to the "Links" tab and open the "Label" section.
  4. Using theSourceLink-Picker, select the previously createdYInput.

That's it! You can now test this functionality by entering the preview mode and typing text in theYInput component. The YLabel component should update accordingly.

Final Result:

tip

If you want to include text before or after the input value, you can add it in the Style > Label > Label property of the YLabel component. To display the input value, use {value} as a placeholder.
