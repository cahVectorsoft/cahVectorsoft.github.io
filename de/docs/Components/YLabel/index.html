Number Property

The identifier of the component, which is unique inside a page.

Not Editable
String Property
---y-label__component-type

The type of the component. For this component it is y-label.

Name: name
String Property
---y-label__name

The custom name of the component. For better identification of the component.

Name: version
String Property
---y-label__version

The custom version of the Label-component. This can be used to ensure that all components work well together.

Name: coreTheme
String Property
---y-label__core-theme

The core-theme, which will be apllied to the Label. For further information on themes visit the theme-manager page.

Name: subTheme
String Property
---y-label__sub-theme

The sub-theme subordinated to the previously specified core-theme, which will be apllied to the Label. For further information on themes visit the theme-manager page.

Name: groupTheme
String Property
---y-label__group-theme

??? REVISION PLS !!! The group-theme which is specified especially for the Label-component inside the sub-theme. For further information on themes visit the theme-manager page.

Name: display
List Property
---y-label__display

This property specifies the display behavior of the component. This can be be set to:

  • none
  • block
  • flex
  • inline
Name: position
List Property
---y-label__position

This property specifies the type of positioning method used for the component. This can be be set to:

  • static
  • relative
  • absolute
  • sticky
  • fixed
Name: visible
Boolean Property
---y-label__visible

This property can toggle the visibility of the component. The two modes are completely hidden and fully shown.
