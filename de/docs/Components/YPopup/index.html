String Property

The custom name of the component. For better identification of the component.

String Property
---y-popup__version

The custom version of the Popup-component. This can be used to ensure that all components work well together.

Name: coreTheme
String Property
---y-popup__core-theme

The core-theme, which will be apllied to the Popup. For further information on themes visit the themes page.

Name: subTheme
String Property
---y-popup__sub-theme

The sub-theme subordinated to the previously specified core-theme, which will be apllied to the Popup. For further information on themes visit the themes page.

Name: groupTheme
String Property
---y-popup__group-theme

The group-theme is a further variation of the subtheme which is specified especially for variations of a component inside the sub-theme. For further information on themes visit the Theme-Manager page.
