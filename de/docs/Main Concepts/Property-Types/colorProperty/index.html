Color Property

Description​

The color property can simply be nderstood as three pairs of 2-digit hexadecimal numbers. A fourth pair can be added for the so called alpha value which represents the opacity of the color. The color property is mostly used for setting colors of elements, like background colors or font colors. Alternatively the color property can also be set to the value transparent.

can be theme injectable:true
Possible input characters:#000000 - #FFFFFF or #00000000 - #FFFFFFFF or transparent
