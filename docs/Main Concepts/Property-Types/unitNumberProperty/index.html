Unit Number Property

Description​

The unit number property can simply be understood as a combination of the Number Property as the value and the List Property as the unit. The unit number property is used for element settings that describe sizes with varying units, font sizes, widths and heights.

can be theme injectable:true
Possible input characters:0-9 and A-Z, a-z, 0-9, Special characters
