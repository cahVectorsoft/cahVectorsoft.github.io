String Property

String Property

String Property
---y-image__sub-theme

The sub-theme subordinated to the previously specified core-theme, which will be apllied to the Image. For further information on themes visit the theme-manager page.

String Property

The url to the source of the file to be set as the image of the Image-component. This can be set to a text containing a valid url path. The path should not be to your local files (starting with e.g. C:/...), but instead to the web (starting with e.g. https://www....).
