Explorer

The Explorer has three sections that can be accessed through tabs or the More Button.

Components​

In Components, we find all the components available for creating the UI. To do this, we select a container component and can then add the component by clicking on the component icon. Alternatively, components can also be added to a page via drag & drop using the component icon.

Layout​

Under Layouts, you'll find various pre-made layout templates. The group of layout modules offers simple foundational templates. These can be combined to create larger, more complex layouts.

Page layouts are templates that include entire forms or pages.

Overview​

Here, all popups and pages of the project are listed. The active page or popup is loaded in the work area.
