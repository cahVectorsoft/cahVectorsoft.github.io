Number

Description​

The number property can simply be understood as a number. The number property is mostly used for element settings that describe size or quantity. Examples are quantity, unitless sizes or sizes with predefined units, like font weight or shadow spread.

FeatureValue
Theme injectable✓
Possible Inputs0-9

Occurrences​

Toolbar​

Here the number property can be found in numerous Toolbar Extensions.

Detail Panel​

Inside the Detail Panel the number property is used several times in various pickers, for example in the Detail Panel Number Picker.
